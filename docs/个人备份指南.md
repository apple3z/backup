## 🔄 备份方式

### 方式 1: GitHub 自动备份 (推荐)

瑞贝卡已经配置好自动备份到 GitHub:
- **仓库**: https://github.com/apple3z/backup
- **内容**: 所有配置文件 + 工作区文件 + 记忆文件
- **更新**: 每次重大变更后手动推送

**查看备份:**
```bash
# 在浏览器访问
https://github.com/apple3z/backup
```

### 方式 2: 本地手动备份

```bash
# 1. 创建备份目录
mkdir -p ~/backups/openclaw/$(date +%Y%m%d)

# 2. 复制配置文件
cp -r ~/.openclaw/ ~/backups/openclaw/$(date +%Y%m%d)/

# 3. 复制技能库
cp -r ~/.agents/skills/ ~/backups/openclaw/$(date +%Y%m%d)/skills/

# 4. 压缩备份
cd ~/backups/openclaw/
tar -czf openclaw-backup-$(date +%Y%m%d).tar.gz $(date +%Y%m%d)/
```

### 方式 3: 外部存储备份

```bash
# 备份到外部硬盘/U 盘
cp -r ~/.openclaw/ /mnt/usb-drive/openclaw-backup/
cp -r ~/.agents/skills/ /mnt/usb-drive/openclaw-backup/skills/
```

## 🚨 恢复流程

### 从 GitHub 恢复

```bash
# 1. 克隆备份仓库
cd /tmp
git clone https://github.com/apple3z/backup.git openclaw-backup
cd openclaw-backup

# 2. 停止 Gateway
openclaw gateway stop

# 3. 恢复配置
cp config/openclaw.json.* ~/.openclaw/openclaw.json

# 4. 恢复工作区
cp -r workspace/* ~/.openclaw/workspace/

# 5. 重启 Gateway
openclaw gateway restart

# 6. 重新配置敏感信息
# 编辑 openclaw.json，填入 API Keys、Token 等
```

### 从本地备份恢复

```bash
# 1. 停止 Gateway
openclaw gateway stop

# 2. 恢复备份
cp -r ~/.openclaw/backup/config/openclaw.json.* ~/.openclaw/openclaw.json
cp -r ~/.openclaw/backup/workspace/* ~/.openclaw/workspace/

# 3. 重启 Gateway
openclaw gateway restart
```

### 完全重装

```bash
# 1. 重新安装 OpenClaw
npm install -g openclaw@latest

# 2. 恢复配置
cp ~/backups/openclaw/YYYYMMDD/openclaw.json ~/.openclaw/openclaw.json

# 3. 恢复工作区
cp -r ~/backups/openclaw/YYYYMMDD/workspace/* ~/.openclaw/workspace/

# 4. 重新安装技能
# 从技能清单安装
cat ~/backups/openclaw/YYYYMMDD/skills/installed-skills.txt

# 5. 配置敏感信息
# 编辑 openclaw.json

# 6. 启动 Gateway
openclaw gateway start
```
